A full-stack web application that helps users make smarter investment decisions through real-time data and intelligent recommendations.

Engineered a recommendation system that analyzes historical market data and trends to suggest suitable stocks and assets.

* Built responsive, interactive dashboards for visualizing live market data.
* Developed the complete stack — frontend (React, JavaScript, HTML/CSS), backend (Node.js, Express), and database (MongoDB).
* Integrated Axios for API communication and React Router DOM for smooth client-side navigation.
* Configured CORS policies to enable secure, cross-origin data exchange between frontend and backend.
